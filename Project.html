1 Problem Statement

Love Letter is a card game that can be played with 2-4 players. I would be creating a Love Letter web based game that includes the use of Artificial Intelligence.

2 Problem Description

Over the past year I found a love for card games. Love letter is a game that involves chance, memory, and risk. To play this game each player is dealt two cards. There are 16 cards total and each card has a special way it can be used and a point value. The whole objective is to either eliminate people using the special instructions of each card or be that last one left with the highest point valued card. Each turn has many different possibilities which is why creating an AI would be both challenging and fun.

3 Project Goals

My original goals for this project are:

  • Create a two player game of Love Letter
  • Integrate an AI that can make possible and smart plays
  • Use different web based languages
  • Use different object oriented design patterns
  • Create a database of users
  • Save each game and load a game
  • Main menu

4 Project Scope

In order to create this game using web languages I will be using HTML and CSS to set up the pages. Java script will be used for the interaction and driving force behind how the game will actually play. PHP will be used for the sever side and will help with the cross platform applications. For my database I will be using MYSQL.

5 Success Factors and Benefits

My ultimate goal is take what I learned from advanced web programming and software Engineering and blend those two classes. I created a game in Advanced Web but I would like to apply the knowledge and skills of good programming practices and object oriented design pattern. I believe my success should be measured on my effort and how well the game plays out. The benefits is that I get to apply my knowledge from one class and apply it on something separate. I also think that the professors of these classes might be able to include some of the things I do into their curriculum.
